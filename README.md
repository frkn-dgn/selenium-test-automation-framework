# Selenium Test Automation Framework

This project is a basic web test automation framework built with **Java, Selenium WebDriver, TestNG, and Maven**.

## Features

- Page Object Model (POM)
- BaseTest for driver setup
- ConfigReader for configuration management
- Screenshot capture on test failure
- Automated test scenarios

## Technologies

- Java
- Selenium WebDriver
- TestNG
- Maven

## Test Scenarios

- Login Test
- Invalid Login Test
- Add to Cart Test
